## Módulo 10: Expresiones Naturales

### 🎯 Objetivo
Usar expresiones que hacen tu inglés sonar más natural y fluido.

### 🧩 Chunks: Expresar Opinión

| Expresión | Uso |
|-----------|-----|
| **I think so** | Creo que sí |
| **I don't think so** | Creo que no |
| **I'm not sure** | No estoy seguro |
| **Maybe** | Quizás |
| **Probably** | Probablemente |
| **I guess** | Supongo |

### 🧩 Chunks: Acuerdo y Desacuerdo

| Expresión | Uso |
|-----------|-----|
| **Of course** | Por supuesto |
| **Sure** | Claro |
| **Definitely** | Definitivamente |
| **Not really** | No realmente |
| **I don't know** | No sé |
| **That's true** | Es verdad |

### 🧩 Chunks: Reacciones

| Expresión | Uso |
|-----------|-----|
| **Really?** | ¿En serio? |
| **That's great!** | ¡Qué bien! |
| **That's too bad** | Qué lástima |
| **No way!** | ¡No puede ser! |
| **Wow!** | ¡Guau! |
| **Oh, I see** | Ah, ya veo |

### 🧩 Chunks: Expresiones de Tiempo

| Expresión | Uso |
|-----------|-----|
| **Right now** | Ahora mismo |
| **In a minute** | En un minuto |
| **Later** | Luego |
| **Soon** | Pronto |
| **Not yet** | Todavía no |
| **Already** | Ya |

### 🧩 Chunks: Expresiones Útiles

| Expresión | Uso |
|-----------|-----|
| **By the way** | Por cierto |
| **Anyway** | De todos modos |
| **Actually** | En realidad |
| **Basically** | Básicamente |
| **You know** | Sabes |
| **I mean** | O sea / Quiero decir |

### 🎧 Listening: Conversación Natural

> A: Hey, are you coming to the party?  
> B: **I'm not sure. Maybe.**  
> A: Come on, it'll be fun!  
> B: **I guess** I could go. What time?  
> A: Around 8.  
> B: **Okay, sure.** See you there!

### 🗣️ Speaking: Usa las Expresiones

Responde usando expresiones naturales:
1. "Is it going to rain?" → **Maybe** / **I don't think so**
2. "Do you like pizza?" → **Of course!** / **Definitely**
3. "Did you finish?" → **Not yet** / **Already**
4. "She got a new job!" → **Really? That's great!**

### 🔁 Práctica: Completa el Diálogo

|--------|------------|

| Want to go out? | _____________ (no estoy seguro) |
| I got promoted! | _____________ (¡qué bien!) |
| Is he coming? | _____________ (probablemente) |

## Módulo 11: Colocaciones Básicas

### 🎯 Objetivo
Aprender combinaciones de palabras que siempre van juntas en inglés.

### 🧠 ¿Qué es una colocación?

Una **colocación** es una combinación de palabras que los nativos usan naturalmente. No se traducen literalmente.

| ❌ Incorrecto | ✅ Correcto |
|---------------|-------------|
| do breakfast | **make** breakfast |
| make homework | **do** homework |
| take a decision | **make** a decision |

### 🧩 Colocaciones con HAVE

| Colocación | Significado |
|------------|-------------|
| **have breakfast** | desayunar |
| **have lunch** | almorzar |
| **have dinner** | cenar |
| **have a drink** | tomar algo |
| **have fun** | divertirse |
| **have a good time** | pasarla bien |

### 🧩 Colocaciones con MAKE

| Colocación | Significado |
|------------|-------------|
| **make money** | ganar dinero |
| **make a mistake** | cometer un error |
| **make a decision** | tomar una decisión |
| **make a phone call** | hacer una llamada |
| **make plans** | hacer planes |
| **make friends** | hacer amigos |

### 🧩 Colocaciones con DO

| Colocación | Significado |
|------------|-------------|
| **do homework** | hacer tarea |

| **do laundry** | lavar la ropa |

| **do a favor** | hacer un favor |

### 🧩 Colocaciones con TAKE

| Colocación | Significado |
|------------|-------------|
| **take a break** | tomar un descanso |

| **take a photo** | tomar una foto |
| **take a nap** | tomar una siesta |
| **take notes** | tomar notas |
| **take your time** | tomarse su tiempo |

### 🧩 Colocaciones con GO

| Colocación | Significado |
|------------|-------------|
| **go home** | ir a casa |
| **go to work** | ir al trabajo |
| **go shopping** | ir de compras |
| **go for a walk** | ir a caminar |
| **go out** | salir |
| **go to bed** | ir a dormir |

### 🎧 Listening: Colocaciones en Contexto

> "Every morning I **have breakfast** at 7. Then I **take a shower** and **go to work**. At lunch, I **take a break** and **have lunch** with my coworkers. After work, I **go home** and **do the dishes**. Sometimes I **go for a walk** before I **go to bed**."

### 🗣️ Speaking: Practica las Colocaciones

Di estas frases:

2. **I need to make a phone call**
3. **Can you do me a favor?**
4. **Let's take a break**
5. **I'm going home now**
