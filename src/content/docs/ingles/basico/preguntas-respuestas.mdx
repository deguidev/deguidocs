## Módulo 12: Preguntas WH

### 🎯 Objetivo
Dominar las preguntas con palabras WH para obtener información específica.

### 🧠 Las Palabras WH

| Palabra | Significado | Pregunta por |
|---------|-------------|--------------|
| **What** | Qué | Cosas, acciones |
| **Where** | Dónde | Lugares |
| **When** | Cuándo | Tiempo |
| **Who** | Quién | Personas |
| **Why** | Por qué | Razones |
| **How** | Cómo | Manera, estado |

## Módulo 13: Preguntas Cerradas (Yes/No)

### 🎯 Objetivo
Hacer preguntas que se responden con Sí o No usando auxiliares.

### 🧠 Estructura de Preguntas Yes/No

| Auxiliar | Estructura | Ejemplo |
|----------|------------|---------|
| **BE** | Am/Is/Are + sujeto...? | Are you ready? |
| **DO** | Do/Does + sujeto + verbo...? | Do you like coffee? |
| **CAN** | Can + sujeto + verbo...? | Can you help me? |

### 🧩 Chunks: Preguntas con BE

| Pregunta | Uso |
|----------|-----|
| **Are you ready?** | ¿Estás listo? |
| **Is this correct?** | ¿Esto es correcto? |
| **Are you sure?** | ¿Estás seguro? |
| **Is it far?** | ¿Está lejos? |
| **Are you coming?** | ¿Vienes? |
| **Is everything okay?** | ¿Todo bien? |

### 🧩 Chunks: Preguntas con DO/DOES

| Pregunta | Uso |
|----------|-----|
| **Do you understand?** | ¿Entiendes? |
| **Do you speak English?** | ¿Hablas inglés? |
| **Do you have time?** | ¿Tienes tiempo? |
| **Does it work?** | ¿Funciona? |
| **Do you want some?** | ¿Quieres un poco? |
| **Do you need help?** | ¿Necesitas ayuda? |

### 🧩 Chunks: Preguntas con CAN

| Pregunta | Uso |
|----------|-----|
| **Can you help me?** | ¿Puedes ayudarme? |
| **Can I ask you something?** | ¿Puedo preguntarte algo? |
| **Can you repeat that?** | ¿Puedes repetir eso? |
| **Can I have...?** | ¿Me das...? |
| **Can we go?** | ¿Podemos ir? |

### 🧩 Chunks: Otras Preguntas Comunes

| Pregunta | Uso |
|----------|-----|
| **Would you like...?** | ¿Te gustaría...? |
| **Have you ever...?** | ¿Alguna vez has...? |
| **Did you...?** | ¿Hiciste...? |
| **Will you...?** | ¿Vas a...? |

### 🎧 Listening: Diálogo con Preguntas Yes/No

> A: **Do you speak English?**  
> B: Yes, I do. A little.  
> A: Great! **Can you help me?**  
> B: Sure. What do you need?  
> A: **Is there a bank near here?**  
> B: Yes, there is. It's on Main Street.

### 🗣️ Speaking: Practica las Preguntas

Di estas preguntas:
1. **Do you understand?**
2. **Are you ready?**
3. **Can you help me?**
4. **Do you have time?**
5. **Is this correct?**

|------------|----------|
| You are tired | **Are you tired?** |
| She speaks English | **Does she speak English?** |
| They can come | **Can they come?** |
| It is expensive | **Is it expensive?** |

## Módulo 14: Respuestas Naturales

### 🎯 Objetivo
Responder de forma natural, no solo con "Yes" o "No".

### 🧠 Respuestas Cortas

| Pregunta | Respuesta Sí | Respuesta No |
|----------|--------------|--------------|
| Are you...? | Yes, I am | No, I'm not |
| Is he/she...? | Yes, he/she is | No, he/she isn't |
| Do you...? | Yes, I do | No, I don't |
| Does he/she...? | Yes, he/she does | No, he/she doesn't |
| Can you...? | Yes, I can | No, I can't |

### 🧩 Chunks: Respuestas Afirmativas Naturales

| En lugar de solo "Yes" | Di |
|------------------------|-----|
| Yes | **Yeah** / **Yep** / **Sure** |
| Yes, of course | **Of course** / **Definitely** |
| Yes, I think so | **I think so** / **Probably** |
| Yes, a little | **A little** / **Kind of** |

### 🧩 Chunks: Respuestas Negativas Naturales

| En lugar de solo "No" | Di |
|-----------------------|-----|
| No | **Nope** / **Not really** |
| No, I don't think so | **I don't think so** |
| No, not yet | **Not yet** |
| No, I'm not sure | **I'm not sure** |

### 🧩 Chunks: Respuestas Extendidas

| Pregunta | Respuesta corta | Respuesta extendida |
|----------|-----------------|---------------------|
| Do you like pizza? | Yes, I do | **Yeah, I love it!** |
| Are you coming? | Yes, I am | **Yeah, I'll be there** |
| Can you help? | Yes, I can | **Sure, no problem** |
| Is it far? | No, it isn't | **No, it's pretty close** |

### 🧩 Chunks: Respuestas Inciertas

| Expresión | Uso |
|-----------|-----|
| **Maybe** | Quizás |
| **I'm not sure** | No estoy seguro |
| **I don't know** | No sé |
| **It depends** | Depende |
| **I think so** | Creo que sí |
| **I hope so** | Espero que sí |

### 🎧 Listening: Respuestas Naturales

> A: Do you want to go to the movies?  
> B: **Yeah, sure!** What time?  
> A: Around 7. Is that okay?  
> B: **I think so.** Let me check.  
> A: Can you drive?  
> B: **Not really,** my car is broken.

### 🗣️ Speaking: Practica Respuestas

Responde de forma natural:
1. "Do you like coffee?" → **Yeah, I love it!**
2. "Are you busy?" → **Kind of** / **Not really**
3. "Can you come tomorrow?" → **I think so** / **I'm not sure**
4. "Is it expensive?" → **Yeah, pretty much** / **Not really**
