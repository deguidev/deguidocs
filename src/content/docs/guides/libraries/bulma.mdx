## Integración Básica <Badge text="Paso 2" variant="note" />

### HTML
```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="path/to/bulma.min.css">
</head>
<body>
  <div class="container">
    <button class="button is-primary">Botón Bulma</button>
  </div>
</body>
</html>
```

### Vue.js
```vue
<!-- App.vue -->
<template>
  <div class="container">
    <button class="button is-primary">Botón Bulma</button>
  </div>
</template>

<style>
@import 'bulma/css/bulma.min.css';
</style>
```

### Angular

Primero, importa los estilos en tu archivo CSS global:
```css
/* styles.css */
@import 'bulma/css/bulma.min.css';
```

Luego, crea tu plantilla HTML:
```html
<!-- app.component.html -->
<div class="container">
  <button class="button is-primary">Botón Bulma</button>
</div>
```

Finalmente, configura Angular para incluir los estilos:
```json
{
  "projects": {
    "your-app": {
      "architect": {
        "build": {
          "options": {
            "styles": [
              "node_modules/bulma/css/bulma.min.css"
            ]
          }
        }
      }
    }
  }
}
```

## Personalización con SASS <Badge text="Paso 3" variant="note" />

### Estructura de archivos
<Code code={`// styles/custom-bulma.scss

// 1. Importar variables iniciales de Bulma
@import "bulma/sass/utilities/initial-variables";
@import "bulma/sass/utilities/functions";

// 2. Personalizar variables
$primary: #1a365d;
$primary-light: findLightColor($primary);
$primary-dark: findDarkColor($primary);

$link: #485fc7;
$info: #3e8ed0;
$success: #48c78e;
$warning: #ffe08a;
$danger: #f14668;

// 3. Personalizar componentes
$body-background-color: #fafafa;
$body-size: 16px;

// 4. Importar el resto de Bulma
@import "bulma/bulma";`} lang="scss" />

## Optimización <Badge text="Paso 5" variant="caution" />

### Importar solo lo necesario
<Code code={`// styles/minimal-bulma.scss

// Base
@import "bulma/sass/base/minireset";
@import "bulma/sass/base/generic";

// Components que necesitas
@import "bulma/sass/elements/button";
@import "bulma/sass/elements/container";
@import "bulma/sass/elements/title";
@import "bulma/sass/form/_all";
@import "bulma/sass/components/navbar";

```html
<!-- Espaciado -->
<div class="mt-4 mb-4 px-2">
  Contenido con márgenes y padding
</div>

<!-- Flexbox -->

  Contenido centrado
</div>

<!-- Responsive helpers -->

  Solo visible en tablet y desktop
</div>
```
