**Flexbox** (Flexible Box Layout) es un sistema de diseño unidimensional que facilita la distribución de elementos en filas o columnas de manera flexible y eficiente.

<Aside type="tip">
🎯 **Concepto clave:** Flexbox revolucionó el diseño web al hacer que tareas comunes (centrar elementos, distribuir espacio, alinear contenido) sean extremadamente simples.
</Aside>

---

## 🚀 Introducción a Flexbox

### ¿Qué es Flexbox?

Flexbox es un modelo de layout que permite:
- ✅ Distribuir espacio entre elementos
- ✅ Alinear elementos fácilmente
- ✅ Crear layouts responsivos sin media queries
- ✅ Reordenar elementos visualmente
- ✅ Manejar tamaños dinámicos

### Activar Flexbox

```css
.contenedor {
  display: flex;  /* Activa Flexbox */
}

/* O en línea */
.contenedor-inline {
  display: inline-flex;  /* Flexbox inline */
}
```

<Aside type="note">
💡 **Importante:** Al aplicar `display: flex` a un contenedor:
- El contenedor se convierte en **flex container**
- Sus hijos directos se convierten en **flex items**
- Solo los hijos directos son afectados (no los nietos)
</Aside>

## 📐 Ejes en Flexbox

Flexbox trabaja con **dos ejes** perpendiculares:

### Eje Principal (Main Axis) y Eje Secundario (Cross Axis)

```plaintext
flex-direction: row (default)
┌─────────────────────────────────────┐
│                                     │
│  ┌────┐  ┌────┐  ┌────┐  ┌────┐   │
│  │ 1  │  │ 2  │  │ 3  │  │ 4  │   │ ← Cross Axis
│  └────┘  └────┘  └────┘  └────┘   │
│  ────────────────────────────────→ │
│         Main Axis                   │
└─────────────────────────────────────┘

flex-direction: column
┌─────────────────┐
│                 │
│  ┌──────────┐  │ ↓
│  │    1     │  │ │
│  └──────────┘  │ │
│  ┌──────────┐  │ │ Main Axis
│  │    2     │  │ │
│  └──────────┘  │ │
│  ┌──────────┐  │ │
│  │    3     │  │ ↓
│  └──────────┘  │
│  ←───────────→ │
│   Cross Axis   │
└─────────────────┘
```

### `flex-direction` - Define el eje principal

```css
.contenedor {
  display: flex;
  flex-direction: row;           /* Horizontal (default) → */
  flex-direction: row-reverse;   /* Horizontal invertido ← */
  flex-direction: column;        /* Vertical ↓ */
  flex-direction: column-reverse; /* Vertical invertido ↑ */
}
```

### `flex-flow` - Shorthand para direction y wrap

```css
/* Forma larga */
.contenedor {
  flex-direction: row;
  flex-wrap: wrap;
}

/* Forma corta (equivalente) */
.contenedor {
  flex-flow: row wrap;
  /* direction wrap */
}

/* Ejemplos */
.contenedor {
  flex-flow: column nowrap;
  flex-flow: row-reverse wrap;
  flex-flow: column wrap-reverse;
}
```

## 🎯 Alineación en el Eje Principal

### `justify-content` - Distribuye items en el eje principal

Controla cómo se distribuyen los items a lo largo del **main axis**.

```css
.contenedor {
  display: flex;
  justify-content: flex-start;    /* Inicio (default) */
  justify-content: flex-end;      /* Final */
  justify-content: center;        /* Centro */
  justify-content: space-between; /* Espacio entre items */
  justify-content: space-around;  /* Espacio alrededor */
  justify-content: space-evenly;  /* Espacio uniforme */
}
```

## ⬆️ Alineación en el Eje Secundario

### `align-items` - Alinea items en el eje secundario

Controla cómo se alinean los items a lo largo del **cross axis** (en una sola línea).

```css
.contenedor {
  display: flex;
  align-items: stretch;      /* Estirar (default) */
  align-items: flex-start;   /* Inicio */
  align-items: flex-end;     /* Final */
  align-items: center;       /* Centro */
  align-items: baseline;     /* Línea base del texto */
}
```

### Centrado perfecto con Flexbox

```css
/* El método más simple para centrar */
.contenedor {
  display: flex;
  justify-content: center;  /* Centrado horizontal */
  align-items: center;      /* Centrado vertical */
  height: 100vh;
}
```

```plaintext
┌─────────────────────────────┐
│                             │
│                             │
│         ┌─────────┐         │
│         │Centrado │         │
│         └─────────┘         │
│                             │
│                             │
└─────────────────────────────┘
```

<Aside type="tip">
🎯 **Centrado perfecto:** Esta es la forma más simple y efectiva de centrar elementos vertical y horizontalmente. ¡Adiós a los hacks antiguos!
</Aside>

## 📦 Alineación de Múltiples Líneas

### `align-content` - Alinea líneas en el eje secundario

Controla cómo se distribuyen **múltiples líneas** en el cross axis (solo funciona con `flex-wrap: wrap`).

```css
.contenedor {
  display: flex;
  flex-wrap: wrap;  /* Necesario para múltiples líneas */
  align-content: stretch;       /* Estirar (default) */

  align-content: center;        /* Centro */
  align-content: space-between; /* Espacio entre líneas */
  align-content: space-around;  /* Espacio alrededor */
  align-content: space-evenly;  /* Espacio uniforme */
}
```

## 🔧 Propiedades de los Flex Items

### `align-self` - Alineación individual

Sobrescribe `align-items` para un item específico.

```css
.item {
  align-self: auto;       /* Hereda de align-items (default) */
  align-self: flex-start; /* Al inicio */
  align-self: flex-end;   /* Al final */
  align-self: center;     /* Centrado */
  align-self: baseline;   /* Línea base */
  align-self: stretch;    /* Estirar */
}
```

```plaintext
┌─────────────────────────────────┐
│ ┌───┐           ┌───┐           │
│ │ 1 │           │ 3 │           │ ← flex-start
│ └───┘ ┌───┐     └───┘           │
│       │ 2 │           ┌───┐     │
│       │   │           │ 4 │     │ ← center
│       │   │           └───┘     │
│       └───┘                     │
│                       ┌───┐     │
│                       │ 5 │     │ ← flex-end
│                       └───┘     │
└─────────────────────────────────┘
```

```css

  align-items: center;  /* Todos centrados por defecto */

  align-self: flex-end;  /* Este va al final */
}
```

```css
.item {
```

### `flex-basis` - Tamaño base

Define el tamaño **inicial** de un item antes de distribuir el espacio.

```css
.item {
  flex-basis: auto;    /* Basado en width/height (default) */
  flex-basis: 200px;   /* Tamaño base fijo */
  flex-basis: 50%;     /* Porcentaje del contenedor */
  flex-basis: 0;       /* Sin tamaño base */
}
```

<Tabs>
<TabItem label="flex-basis vs width">
```css
/* width - Tamaño fijo */
.item-width {
  width: 200px;
}

/* flex-basis - Tamaño flexible */
.item-basis {
  flex-basis: 200px;
  flex-grow: 1;  /* Puede crecer desde 200px */
}
```

<Aside type="note">
💡 **Diferencia:** `flex-basis` es el tamaño **inicial** antes de aplicar grow/shrink. `width` es más rígido.
</Aside>
</TabItem>

<TabItem label="Ejemplos">
```css
/* Grid de 3 columnas */
.item {
  flex-basis: 33.333%;
  flex-grow: 0;
  flex-shrink: 0;
}

/* Contenido que crece desde un mínimo */
.item {
  flex-basis: 200px;  /* Mínimo 200px */
  flex-grow: 1;       /* Pero puede crecer */
}

/* Sin tamaño base (distribución pura) */
.item {
  flex-basis: 0;
  flex-grow: 1;  /* Todos exactamente iguales */
}
```
</TabItem>
</Tabs>

### `flex` - Shorthand para grow, shrink y basis

```css
/* Sintaxis: flex: grow shrink basis */

/* Valores comunes */
.item {
  flex: 0 1 auto;     /* Default (no crece, sí encoge, auto) */
  flex: 1;            /* = 1 1 0 (crece, encoge, sin base) */
  flex: auto;         /* = 1 1 auto (flexible completo) */
  flex: none;         /* = 0 0 auto (inflexible) */
}
```

<Tabs>
<TabItem label="Valores comunes">
```css
/* flex: 1 - Más usado ⭐ */
.item {
  flex: 1;
  /* = flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
  /* Items de igual tamaño, flexibles */
}

/* flex: auto - Flexible basado en contenido */
.item {
  flex: auto;
  /* = flex-grow: 1, flex-shrink: 1, flex-basis: auto */
  /* Crece/encoge desde su tamaño natural */
}

/* flex: none - Inflexible */
.item {
  flex: none;
  /* = flex-grow: 0, flex-shrink: 0, flex-basis: auto */
  /* Tamaño fijo, no cambia */
}

/* flex: 0 0 200px - Tamaño fijo específico */
.item {
  flex: 0 0 200px;
  /* No crece, no encoge, 200px fijo */
}
```
</TabItem>

<TabItem label="Ejemplos prácticos">
```css
/* Layout típico: Sidebar + Contenido */
.sidebar {
  flex: 0 0 250px;  /* 250px fijo */
}

.contenido {
  flex: 1;  /* Toma el resto del espacio */
}

/* Grid flexible de 3 columnas */
.columna {
  flex: 1;  /* Todas iguales */
  min-width: 200px;  /* Mínimo 200px */
}

/* Botones con tamaño mínimo */
.boton {
  flex: 0 0 auto;  /* No crece ni encoge */
  min-width: 100px;
}

/* Header con logo, nav y acciones */
.logo {
  flex: 0 0 150px;  /* Logo fijo */
}

.nav {
  flex: 1;  /* Nav flexible */
}

.acciones {
  flex: 0 0 auto;  /* Acciones según contenido */
}
```
</TabItem>

<TabItem label="Patrones comunes">

## 🔄 Orden y Reordenamiento

### `order` - Cambiar el orden visual

Controla el orden de aparición de los items **sin cambiar el HTML**.

```css
.item {
  order: 0;   /* Default - orden del HTML */
  order: 1;   /* Aparece después */
  order: -1;  /* Aparece antes */
}
```

<Tabs>
<TabItem label="Cómo funciona">
```css
/* HTML: <div>1</div><div>2</div><div>3</div> */

.item-1 { order: 0; }  /* Default */
.item-2 { order: -1; } /* Va primero */
.item-3 { order: 2; }  /* Va último */
```

```plaintext
HTML:
[1] [2] [3]

Visual con order:
[2] [1] [3]
 ↑   ↑   ↑
-1   0   2
```

Items se ordenan de menor a mayor valor de `order`
</TabItem>

<TabItem label="Ejemplos prácticos">
```css
/* Responsive: Cambiar orden en mobile */
.sidebar {
  order: 2;  /* Sidebar después en mobile */
}

.contenido {
  order: 1;  /* Contenido primero */
}

@media (min-width: 768px) {
  .sidebar {
    order: 1;  /* Sidebar primero en desktop */
  }
  
  .contenido {
    order: 2;
  }
}

/* Destacar item */
.item-destacado {
  order: -1;  /* Siempre primero */
}

/* Mover al final */
.item-final {
  order: 999;  /* Al final */
}
```
</TabItem>

<TabItem label="Casos de uso">
```css
/* 1. Reordenar navegación en mobile */
.nav-item-importante {
  order: -1;  /* Primero en mobile */
}

/* 2. Formulario: Botón submit al final */
.submit-button {
  order: 999;
}

/* 3. Dashboard: Widget importante primero */
.widget-critico {
  order: -10;
}

/* 4. E-commerce: Precio destacado */
.precio {
  order: -1;  /* Precio primero visualmente */
}
```

<Aside type="caution">
⚠️ **Accesibilidad:** `order` solo cambia el orden **visual**. El orden en el DOM (para lectores de pantalla y tabulación) permanece igual. Úsalo con cuidado.
</Aside>
</TabItem>
</Tabs>

## 🎓 Resumen

<Aside type="note">
**Puntos clave de esta lección:**

### Conceptos básicos
✅ **display: flex** - Activa Flexbox en el contenedor  
✅ **Flex container** - El elemento padre  
✅ **Flex items** - Los hijos directos  
✅ **Main axis** - Eje principal (horizontal o vertical)  
✅ **Cross axis** - Eje secundario (perpendicular)  

### Propiedades del contenedor
✅ **flex-direction** - row, column, row-reverse, column-reverse  
✅ **flex-wrap** - nowrap, wrap, wrap-reverse  
✅ **justify-content** - Alineación en main axis ⭐  
✅ **align-items** - Alineación en cross axis ⭐  
✅ **align-content** - Alineación de líneas múltiples  
✅ **gap** - Espacio entre items  

### Propiedades de los items
✅ **flex-grow** - Capacidad de crecimiento (0 = no crece)  
✅ **flex-shrink** - Capacidad de encogimiento (0 = no encoge)  
✅ **flex-basis** - Tamaño base inicial  
✅ **flex** - Shorthand: grow shrink basis ⭐  
✅ **align-self** - Alineación individual  
✅ **order** - Cambiar orden visual  

### Valores más usados
✅ **flex: 1** - Item flexible de igual tamaño  
✅ **flex: 0 0 auto** - Tamaño fijo según contenido  
✅ **justify-content: space-between** - Distribuir con espacio  
✅ **align-items: center** - Centrar verticalmente  

### Centrado perfecto
```css
.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

**Próximo paso:** En la siguiente lección aprenderás sobre Grid Layout en CSS.
</Aside>
