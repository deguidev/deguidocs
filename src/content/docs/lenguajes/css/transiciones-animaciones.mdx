## üîÑ Transiciones (transition)

### `transition` - Efectos suaves entre estados

Las transiciones crean cambios suaves entre valores de propiedades CSS cuando cambia el estado de un elemento.

```css
.elemento {
  transition: property duration timing-function delay;
  
  /* Ejemplos */
  transition: all 0.3s ease;
  transition: background-color 0.5s ease-in-out;
  transition: transform 0.3s cubic-bezier(0.68, -0.55, 0.265, 1.55);
  
  /* M√∫ltiples propiedades */
  transition: background-color 0.3s, transform 0.5s, opacity 0.2s;
}
```

```css
  transition: all 0.3s ease;
}
```

**Par√°metros:**
- ‚úÖ **property** - Propiedad a animar (all, background, transform, etc.)
- ‚úÖ **duration** - Duraci√≥n (0.3s, 500ms)
- ‚úÖ **timing-function** - Curva de animaci√≥n
- ‚úÖ **delay** - Retraso antes de iniciar (opcional)

## üé¨ Animaciones (@keyframes)

### `animation` y `@keyframes` - Animaciones complejas

Las animaciones permiten crear secuencias de movimiento con m√∫ltiples pasos usando `@keyframes`.

```css
/* Definir la animaci√≥n */
@keyframes nombre-animacion {
  0% {
    transform: translateX(0);
    opacity: 0;
  }
  50% {
    transform: translateX(100px);
    opacity: 1;
  }
  100% {
    transform: translateX(0);
    opacity: 0;
  }
}

/* Aplicar la animaci√≥n */
.elemento {
  animation: nombre-animacion 2s ease-in-out infinite;
}
```

```css
  animation-delay: 0s;                        /* Retraso */
  animation-iteration-count: infinite;        /* Repeticiones (n√∫mero o infinite) */
  animation-direction: normal;                /* Direcci√≥n (normal, reverse, alternate) */
```

- ‚úÖ **delay** - Retraso antes de iniciar
- ‚úÖ **iteration-count** - N√∫mero de repeticiones (infinite)
- ‚úÖ **direction** - Direcci√≥n (normal, reverse, alternate)
- ‚úÖ **fill-mode** - Estado final (forwards, backwards, both)

## ‚ôæÔ∏è Animaciones Infinitas y por Pasos

### Animaciones infinitas

Animaciones que se repiten continuamente usando `animation-iteration-count: infinite`.

```css
.elemento {
  animation: nombre 2s ease-in-out infinite;
  /* Se repite infinitamente */
}

/* Animaci√≥n alternada (va y viene) */
.elemento {
  animation: nombre 2s ease-in-out infinite alternate;
  /* Va hacia adelante y luego hacia atr√°s */
}
```

### Animaciones por pasos (steps)

Animaciones que avanzan en pasos discretos en lugar de suaves.

```css
.elemento {
  animation: nombre 1s steps(4) infinite;
  /* Avanza en 4 pasos discretos */
}

/* √ötil para sprites */
.sprite {
  animation: sprite-walk 0.8s steps(8) infinite;
}

@keyframes sprite-walk {
  from { background-position: 0 0; }
  to { background-position: -800px 0; }
}
```
