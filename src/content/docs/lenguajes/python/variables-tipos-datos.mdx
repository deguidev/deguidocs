## 📦 2.1 ¿Qué es una variable y cómo se crea?

<Tabs>
<TabItem label="👔 Parte Teórica">

### Concepto de variable

Una **variable** es un espacio de memoria identificado por un nombre que almacena un valor que puede cambiar durante la ejecución del programa. En términos simples, es como una caja etiquetada donde guardas información para usarla más tarde.

### Características de las variables en Python

**🔹 Asignación dinámica**  
En Python no es necesario declarar el tipo de dato explícitamente. El intérprete infiere automáticamente el tipo según el valor asignado.

**🔹 Tipado dinámico**  
Una variable puede cambiar de tipo durante la ejecución del programa. Puedes asignar un número y luego reasignar texto a la misma variable.

**🔹 Referencias a objetos**  
Las variables en Python son referencias a objetos en memoria, no contenedores directos de valores.

### Sintaxis de creación

La sintaxis para crear una variable es:

```python
nombre_variable = valor
```

El operador `=` se llama **operador de asignación** y asigna el valor de la derecha a la variable de la izquierda.

### Propósito de las variables

- **Almacenar datos**: Guardar información para uso posterior
- **Reutilización**: Usar el mismo valor múltiples veces sin repetirlo
- **Legibilidad**: Dar nombres significativos a los valores
- **Cálculos**: Realizar operaciones y guardar resultados

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Creación básica de variables

## 🎯 2.2 Tipos de datos: texto, números y booleanos

<Tabs>
<TabItem label="👔 Parte Teórica">

### Tipos de datos primitivos en Python

Python incluye varios tipos de datos fundamentales que representan diferentes categorías de información.

### 📝 Cadenas de texto (str)

Las **cadenas** o **strings** representan secuencias de caracteres (texto). Se definen entre comillas simples `'...'` o dobles `"..."`.

**Características:**
- Inmutables: no se pueden modificar después de creadas
- Soportan operaciones como concatenación y repetición
- Pueden contener letras, números, símbolos y espacios

### 🔢 Números enteros (int)

Los **enteros** representan números sin parte decimal. Pueden ser positivos, negativos o cero.

**Características:**
- Precisión arbitraria: pueden ser tan grandes como la memoria lo permita
- Soportan operaciones aritméticas estándar
- No tienen límite de tamaño (a diferencia de otros lenguajes)

### 🔢 Números decimales (float)

Los **flotantes** representan números con parte decimal (punto flotante).

**Características:**
- Usan notación de punto decimal: `3.14`
- Soportan notación científica: `1.5e-3` (0.0015)
- Tienen precisión limitada (aproximadamente 15-17 dígitos)

### ✅ Booleanos (bool)

Los **booleanos** representan valores de verdad: `True` (verdadero) o `False` (falso).

**Características:**
- Solo dos valores posibles: `True` y `False`
- Se usan en condiciones y expresiones lógicas
- Son subclase de int: `True` equivale a 1, `False` a 0

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Booleanos (bool)

<Code code={`# Valores booleanos

### Comparación de tipos

<Code code={`# Diferentes tipos de datos

## 📋 2.3 Reglas para nombrar variables correctamente

<Tabs>
<TabItem label="👔 Parte Teórica">

### Reglas obligatorias (sintaxis)

Python impone reglas estrictas que deben cumplirse para que un nombre de variable sea válido:

**✅ Regla 1: Caracteres permitidos**  
Los nombres solo pueden contener letras (a-z, A-Z), números (0-9) y guiones bajos (_).

**✅ Regla 2: Primer carácter**  
Debe comenzar con una letra o guión bajo, nunca con un número.

**✅ Regla 3: Sensibilidad a mayúsculas**  
Python distingue entre mayúsculas y minúsculas. `edad`, `Edad` y `EDAD` son tres variables diferentes.

**✅ Regla 4: Palabras reservadas**  
No se pueden usar palabras clave de Python como nombres de variables (`if`, `for`, `while`, `class`, `def`, etc.).

**✅ Regla 5: Sin espacios**  
Los nombres no pueden contener espacios. Usa guiones bajos para separar palabras.

### Convenciones recomendadas (estilo PEP 8)

Además de las reglas obligatorias, existen convenciones de estilo que mejoran la legibilidad:

**🔹 snake_case para variables**  
Usa minúsculas con guiones bajos para separar palabras: `nombre_completo`, `edad_usuario`, `precio_total`.

**🔹 Nombres descriptivos**  
Elige nombres que expliquen claramente qué contiene la variable: `temperatura_maxima` en lugar de `temp` o `t`.

**🔹 Evitar abreviaturas confusas**  
Prefiere `numero_estudiantes` sobre `num_est`, a menos que la abreviatura sea universalmente conocida.

**🔹 Constantes en MAYÚSCULAS**  
Para valores que no cambian, usa mayúsculas: `PI = 3.14159`, `MAX_INTENTOS = 3`.

**🔹 Evitar nombres de una letra**  
Excepto en bucles (`i`, `j`, `k`) o coordenadas (`x`, `y`, `z`), usa nombres más descriptivos.

## 🔍 2.4 Función type() y comprobación de tipos

<Tabs>
<TabItem label="👔 Parte Teórica">

### La función type()

La función **`type()`** es una función integrada de Python que retorna el tipo de dato de un objeto. Es fundamental para verificar qué tipo de información contiene una variable.

**Sintaxis:**
```python
type(objeto)
```

**Retorna:** La clase del objeto (su tipo de dato)

### Propósito de type()

**🔹 Depuración de código**  
Permite identificar errores relacionados con tipos de datos incompatibles.

**🔹 Validación de datos**  
Verifica que una variable tenga el tipo esperado antes de realizar operaciones.

**🔹 Aprendizaje**  
Ayuda a comprender cómo Python clasifica diferentes valores.

### Tipos de datos comunes retornados

- `<class 'int'>`: Número entero
- `<class 'float'>`: Número decimal
- `<class 'str'>`: Cadena de texto
- `<class 'bool'>`: Booleano
- `<class 'list'>`: Lista
- `<class 'NoneType'>`: Valor None (nulo)

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Uso básico de type()

<Code code={`# Verificar el tipo de diferentes valores
numero = 42
texto = "Python"
decimal = 3.14
booleano = True
nulo = None

print(type(numero))    # <class 'int'>
print(type(texto))     # <class 'str'>
print(type(decimal))   # <class 'float'>
print(type(booleano))  # <class 'bool'>
print(type(nulo))      # <class 'NoneType'>`} lang="python" title="type() básico" />

### Uso de isinstance()

<Code code={`# isinstance() para verificar tipos
edad = 25
nombre = "Ana"
altura = 1.65

# Verificar si es de un tipo específico
print(isinstance(edad, int))      # True
print(isinstance(nombre, str))    # True
print(isinstance(altura, float))  # True
print(isinstance(edad, str))      # False

# Verificar múltiples tipos
numero = 42
print(isinstance(numero, (int, float)))  # True`} lang="python" title="isinstance()" />

## 🔄 2.5 Conversiones entre tipos (int(), str(), float(), etc.)

<Tabs>
<TabItem label="👔 Parte Teórica">

### Conversión de tipos (Type Casting)

La **conversión de tipos** o **casting** es el proceso de transformar un valor de un tipo de dato a otro. Python ofrece funciones integradas para realizar estas conversiones.

### Funciones de conversión principales

**🔹 int() - Convertir a entero**  
Convierte un valor a número entero. Trunca la parte decimal de los flotantes.

**🔹 float() - Convertir a flotante**  
Convierte un valor a número decimal (punto flotante).

**🔹 str() - Convertir a cadena**  
Convierte cualquier valor a su representación textual.

**🔹 bool() - Convertir a booleano**  
Convierte un valor a True o False según reglas específicas.

### Reglas de conversión

**Conversión a int:**
- Strings numéricos: `"123"` → `123`
- Flotantes: `3.14` → `3` (trunca decimales)
- Booleanos: `True` → `1`, `False` → `0`

**Conversión a bool:**
- Valores "falsos": `0`, `0.0`, `""` (string vacío), `None` → `False`
- Cualquier otro valor → `True`

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Conversión a int()

<Code code={`# Convertir string a entero
texto_numero = "123"
numero = int(texto_numero)
print(numero)        # 123
print(type(numero))  # <class 'int'>

# Convertir float a int (trunca decimales)
decimal = 3.99
entero = int(decimal)
print(entero)  # 3 (no redondea, solo trunca)

# Convertir booleano a int
print(int(True))   # 1
print(int(False))  # 0`} lang="python" title="Conversión a int" />

### Conversión a float()

<Code code={`# Convertir string a float
texto = "3.14"
decimal = float(texto)
print(decimal)        # 3.14
print(type(decimal))  # <class 'float'>

# Convertir int a float
entero = 10
flotante = float(entero)
print(flotante)  # 10.0

# Convertir booleano a float
print(float(True))   # 1.0
print(float(False))  # 0.0`} lang="python" title="Conversión a float" />

### Conversión a str()

<Code code={`# Convertir número a string
edad = 25
texto_edad = str(edad)
print(texto_edad)        # "25"
print(type(texto_edad))  # <class 'str'>

# Concatenar después de convertir
nombre = "Ana"
edad = 25
mensaje = nombre + " tiene " + str(edad) + " años"
print(mensaje)  # Ana tiene 25 años`} lang="python" title="Conversión a str" />

### Aplicación práctica: entrada de usuario

<Code code={`# input() siempre retorna string, hay que convertir

# Solicitar año de nacimiento
anio_nacimiento = input("Ingresa tu año de nacimiento: ")

print(f"Tienes {edad} años")`} lang="python" title="Conversión de input" />
