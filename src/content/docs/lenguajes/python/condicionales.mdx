Los operadores relacionales comparan valores y generan expresiones booleanas:

## üîÑ 5.4 Anidamiento de condiciones

<Tabs>
<TabItem label="üëî Parte Te√≥rica">

### Concepto de anidamiento

El **anidamiento** consiste en colocar una estructura condicional dentro de otra, permitiendo crear l√≥gica m√°s compleja donde las decisiones dependen de m√∫ltiples niveles de evaluaci√≥n.

### Cu√°ndo usar anidamiento

**üîπ Decisiones dependientes**: Una decisi√≥n solo tiene sentido despu√©s de otra  
**üîπ Validaciones en cascada**: Verificar m√∫ltiples criterios en secuencia  
**üîπ L√≥gica compleja**: Cuando no puede expresarse con operadores simples  

### Buenas pr√°cticas

- Limitar el anidamiento a 2-3 niveles m√°ximo
- Considerar usar operadores l√≥gicos en lugar de anidar
- Refactorizar si hay m√°s de 3 niveles

</TabItem>

<TabItem label="üí° Ejemplos Pr√°cticos">

### Anidamiento b√°sico

<Code code={`# Anidamiento de dos niveles
