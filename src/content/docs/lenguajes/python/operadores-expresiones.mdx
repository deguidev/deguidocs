### Concepto de operadores aritm√©ticos

Los **operadores aritm√©ticos** son s√≠mbolos que permiten realizar operaciones matem√°ticas sobre valores num√©ricos (enteros y flotantes). Son fundamentales para cualquier c√°lculo o procesamiento de datos num√©ricos en programaci√≥n.

### Operadores aritm√©ticos en Python

**‚ûï Suma (+)**  
Suma dos valores num√©ricos. Tambi√©n se usa para concatenar cadenas de texto.

**‚ûñ Resta (-)**  
Resta el segundo valor del primero. Tambi√©n funciona como operador unario para negar un n√∫mero.

**‚úñÔ∏è Multiplicaci√≥n (*)**  
Multiplica dos valores. Tambi√©n se usa para repetir cadenas y listas.

**‚ûó Divisi√≥n (/)**  
Divide el primer valor entre el segundo. Siempre retorna un n√∫mero flotante (float), incluso si el resultado es entero.

**üî¢ Divisi√≥n entera (//)**  
Divide y retorna solo la parte entera del resultado, descartando los decimales (floor division).

**üìê M√≥dulo (%)**  
Retorna el residuo de la divisi√≥n entre dos n√∫meros. √ötil para determinar si un n√∫mero es par/impar o para operaciones c√≠clicas.

**üî∫ Exponenciaci√≥n (**)**  
Eleva el primer n√∫mero a la potencia del segundo.

### Orden de precedencia

Python sigue el orden matem√°tico est√°ndar (PEMDAS):
1. **Par√©ntesis** `()`
2. **Exponenciaci√≥n** `**`
3. **Multiplicaci√≥n, Divisi√≥n, Divisi√≥n entera, M√≥dulo** `*`, `/`, `//`, `%`
4. **Suma, Resta** `+`, `-`

## ‚öñÔ∏è 3.2 Operadores de comparaci√≥n

<Tabs>
<TabItem label="üëî Parte Te√≥rica">

### Concepto de operadores de comparaci√≥n

Los **operadores de comparaci√≥n** o **relacionales** comparan dos valores y retornan un resultado booleano: `True` si la comparaci√≥n es verdadera, `False` si es falsa. Son esenciales para tomar decisiones en estructuras de control.

### Operadores de comparaci√≥n en Python

**`==` (Igual a)**  
Verifica si dos valores son iguales.

**`!=` (Diferente de)**  
Verifica si dos valores son diferentes.

**`<` (Menor que)**  
Verifica si el primer valor es menor que el segundo.

**`>` (Mayor que)**  
Verifica si el primer valor es mayor que el segundo.

**`<=` (Menor o igual que)**  
Verifica si el primer valor es menor o igual que el segundo.

**`>=` (Mayor o igual que)**  
Verifica si el primer valor es mayor o igual que el segundo.

</TabItem>

<TabItem label="üí° Ejemplos Pr√°cticos">

### Comparaciones b√°sicas

<Code code={`# Igualdad

print(5 == 3)      # False

# Diferencia
print(5 != 3)      # True
print(5 != 5)      # False

# Mayor y menor
print(10 > 5)      # True
print(5 < 10)      # True

# Mayor o igual, menor o igual
print(10 >= 10)    # True
print(5 <= 10)     # True`} lang="python" title="Comparaciones b√°sicas" />

### Aplicaci√≥n en condicionales

## üîÄ 3.3 Operadores l√≥gicos: and, or, not

<Tabs>

### Concepto de operadores l√≥gicos

Los **operadores l√≥gicos** permiten combinar m√∫ltiples expresiones booleanas para crear condiciones m√°s complejas.

### Operadores l√≥gicos en Python

**and (Y l√≥gico)**  
Retorna `True` solo si **ambas** condiciones son verdaderas.

**or (O l√≥gico)**  
Retorna `True` si **al menos una** de las condiciones es verdadera.

**not (Negaci√≥n l√≥gica)**  
Invierte el valor booleano.

</TabItem>

<TabItem label="üí° Ejemplos Pr√°cticos">

### Operador and

## üîÑ 3.4 Operadores de asignaci√≥n compuesta: +=, -=, *=, etc.

<Tabs>
<TabItem label="üëî Parte Te√≥rica">

### Concepto de operadores de asignaci√≥n compuesta

Los **operadores de asignaci√≥n compuesta** combinan una operaci√≥n aritm√©tica con la asignaci√≥n en una sola expresi√≥n. Son atajos que hacen el c√≥digo m√°s conciso.

### Operadores principales

**+= (Suma y asigna)**  
`x += 5` es equivalente a `x = x + 5`

**-= (Resta y asigna)**  
`x -= 3` es equivalente a `x = x - 3`

**\*= (Multiplica y asigna)**  
`x *= 2` es equivalente a `x = x * 2`

**/= (Divide y asigna)**  
`x /= 4` es equivalente a `x = x / 4`

</TabItem>

<TabItem label="üí° Ejemplos Pr√°cticos">

### Operadores compuestos

<Code code={`# += (suma y asigna)

# -= (resta y asigna)

# *= (multiplica y asigna)

precio *= 2
print(precio)  # 100

# /= (divide y asigna)
total = 100
total /= 4
print(total)  # 25.0`} lang="python" title="Operadores compuestos" />

### Aplicaciones pr√°cticas

<Code code={`# Contador de intentos
