### Concepto de bucle

Un **bucle** (o ciclo) es una estructura de control que permite ejecutar un bloque de cÃ³digo repetidamente mientras se cumpla una condiciÃ³n o para cada elemento de una secuencia. Es fundamental para automatizar tareas repetitivas.

**ğŸ”¹ IteraciÃ³n**: Recorre colecciones de datos elemento por elemento

**while**: Repite mientras una condiciÃ³n sea verdadera  
**for**: Itera sobre una secuencia (lista, rango, cadena, etc.)

### Instrucciones de control de flujo

Estas instrucciones modifican el comportamiento normal de los bucles:

### break

**PropÃ³sito**: Termina el bucle inmediatamente  
**Uso**: Cuando se cumple una condiciÃ³n de salida  
**Efecto**: Sale del bucle mÃ¡s interno  

### continue

**PropÃ³sito**: Salta a la siguiente iteraciÃ³n  
**Uso**: Cuando se quiere omitir el resto del cÃ³digo en la iteraciÃ³n actual  
**Efecto**: Vuelve al inicio del bucle  

### pass

**PropÃ³sito**: No hace nada (placeholder)  
**Uso**: Cuando se necesita una sentencia sintÃ¡cticamente pero no se quiere ejecutar nada  
**Efecto**: ContinÃºa normalmente  

</TabItem>
