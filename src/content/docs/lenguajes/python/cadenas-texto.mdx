## üìù 10.1 Qu√© es una cadena (str)

<Tabs>
<TabItem label="üëî Parte Te√≥rica">

### Concepto de cadena (string)

Una **cadena** (string o `str`) es un tipo de dato que representa una secuencia de caracteres. Es uno de los tipos de datos m√°s utilizados en programaci√≥n para trabajar con texto.

### Caracter√≠sticas de las cadenas

**üîπ Inmutables**: No se pueden modificar despu√©s de creadas (se crean nuevas)  
**üîπ Secuenciales**: Los caracteres tienen un orden y posici√≥n (√≠ndice)  
**üîπ Iterables**: Se pueden recorrer car√°cter por car√°cter  
**üîπ Indexables**: Se accede a caracteres mediante √≠ndices (desde 0)  

### Formas de crear cadenas

```python
# Comillas simples
texto1 = 'Hola'

# Comillas dobles
texto2 = "Mundo"

# Comillas triples (multil√≠nea)
texto3 = """Texto
en varias
l√≠neas"""
```

### Prop√≥sito

- **Almacenar texto**: Nombres, mensajes, descripciones
- **Procesar informaci√≥n**: Validar, formatear, transformar
- **Comunicaci√≥n**: Mostrar mensajes al usuario
- **Manipulaci√≥n de datos**: Limpiar, buscar, extraer informaci√≥n

</TabItem>

## üîß 10.2 Operaciones b√°sicas: concatenaci√≥n, repetici√≥n, longitud

### Concatenaci√≥n

La **concatenaci√≥n** une dos o m√°s cadenas en una sola usando el operador `+`.

**Sintaxis**: `cadena1 + cadena2`

### Repetici√≥n

La **repetici√≥n** crea una nueva cadena repitiendo la original usando el operador `*`.

**Sintaxis**: `cadena * numero`

### Longitud

La funci√≥n **`len()`** retorna el n√∫mero de caracteres en una cadena.

**Sintaxis**: `len(cadena)`

### Operadores de comparaci√≥n

Las cadenas se pueden comparar usando operadores relacionales: `==`, `!=`, `<`, `>`, `<=`, `>=` (comparaci√≥n alfab√©tica).

## üî§ 10.3 M√©todos de cadenas: lower(), upper(), replace()

<Tabs>
<TabItem label="üëî Parte Te√≥rica">

### M√©todos de transformaci√≥n

**lower()**: Convierte toda la cadena a min√∫sculas  
**upper()**: Convierte toda la cadena a MAY√öSCULAS  
**capitalize()**: Primera letra en may√∫scula, resto en min√∫sculas  
**title()**: Primera letra de cada palabra en may√∫scula  

### M√©todo replace()

**replace(viejo, nuevo)**: Reemplaza todas las ocurrencias de una subcadena por otra.

**Sintaxis**: `cadena.replace(buscar, reemplazar)`

### M√©todos de limpieza

**strip()**: Elimina espacios al inicio y final  
**lstrip()**: Elimina espacios al inicio  
**rstrip()**: Elimina espacios al final  

### Caracter√≠sticas importantes

**üîπ Retornan nueva cadena**: No modifican la original (inmutabilidad)  
**üîπ Encadenables**: Se pueden usar varios m√©todos seguidos  
**üîπ Case-sensitive**: Distinguen may√∫sculas y min√∫sculas  

</TabItem>

<TabItem label="üí° Ejemplos Pr√°cticos">

## ‚úÇÔ∏è 10.5 Manipulaci√≥n de texto con split() y join()

<Tabs>
<TabItem label="üëî Parte Te√≥rica">

### M√©todo split()

**split(separador)**: Divide una cadena en una lista de subcadenas usando un separador.

**Sintaxis**: `cadena.split(separador)`  
**Por defecto**: Si no se especifica separador, usa espacios en blanco

### M√©todo join()

**join(iterable)**: Une elementos de una lista en una sola cadena usando un separador.

**Sintaxis**: `separador.join(lista)`

### Relaci√≥n entre split() y join()

Son operaciones inversas:
- **split()**: Cadena ‚Üí Lista
- **join()**: Lista ‚Üí Cadena

### Casos de uso

**split()**: Procesar CSV, separar palabras, parsear datos  
**join()**: Construir URLs, crear rutas, formatear salida  

</TabItem>

<TabItem label="üí° Ejemplos Pr√°cticos">
