## 📝 10.1 Qué es una cadena (str)

<Tabs>
<TabItem label="👔 Parte Teórica">

### Concepto de cadena (string)

Una **cadena** (string o `str`) es un tipo de dato que representa una secuencia de caracteres. Es uno de los tipos de datos más utilizados en programación para trabajar con texto.

### Características de las cadenas

**🔹 Inmutables**: No se pueden modificar después de creadas (se crean nuevas)  
**🔹 Secuenciales**: Los caracteres tienen un orden y posición (índice)  
**🔹 Iterables**: Se pueden recorrer carácter por carácter  
**🔹 Indexables**: Se accede a caracteres mediante índices (desde 0)  

### Formas de crear cadenas

```python
# Comillas simples
texto1 = 'Hola'

# Comillas dobles
texto2 = "Mundo"

# Comillas triples (multilínea)
texto3 = """Texto
en varias
líneas"""
```

### Propósito

- **Almacenar texto**: Nombres, mensajes, descripciones
- **Procesar información**: Validar, formatear, transformar
- **Comunicación**: Mostrar mensajes al usuario
- **Manipulación de datos**: Limpiar, buscar, extraer información

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Acceso por índice

<Code code={`texto = "Python"

# Índices negativos (desde el final)

print(texto[-6])  # P (primero desde el final)`} lang="python" title="Índices" />

### Slicing (rebanado)

<Code code={`texto = "Python Programming"

print(texto[::-1])   # gnimmargorP nohtyP (invertida)`} lang="python" title="Slicing" />

## 🔧 10.2 Operaciones básicas: concatenación, repetición, longitud

<Tabs>
<TabItem label="👔 Parte Teórica">

### Concatenación

La **concatenación** une dos o más cadenas en una sola usando el operador `+`.

**Sintaxis**: `cadena1 + cadena2`

### Repetición

La **repetición** crea una nueva cadena repitiendo la original usando el operador `*`.

**Sintaxis**: `cadena * numero`

### Longitud

La función **`len()`** retorna el número de caracteres en una cadena.

**Sintaxis**: `len(cadena)`

### Operadores de comparación

Las cadenas se pueden comparar usando operadores relacionales: `==`, `!=`, `<`, `>`, `<=`, `>=` (comparación alfabética).

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Concatenación

<Code code={`# Unir cadenas con +

print(saludo)  # Hola, Ana!

# Concatenar con números (requiere conversión)
edad = 25
mensaje = nombre + " tiene " + str(edad) + " años"
print(mensaje)  # Ana tiene 25 años`} lang="python" title="Concatenación" />

### Repetición

<Code code={`# Repetir cadenas con *

### Longitud con len()

<Code code={`# Obtener longitud de cadenas

    print("La contraseña debe tener al menos 8 caracteres")

    print("Contraseña válida")

### Comparación de cadenas

<Code code={`# Comparar cadenas

texto2 = "python"

# Igualdad

## 🔤 10.3 Métodos de cadenas: lower(), upper(), replace()

<Tabs>
<TabItem label="👔 Parte Teórica">

### Métodos de transformación

**lower()**: Convierte toda la cadena a minúsculas  
**upper()**: Convierte toda la cadena a MAYÚSCULAS  
**capitalize()**: Primera letra en mayúscula, resto en minúsculas  
**title()**: Primera letra de cada palabra en mayúscula  

### Método replace()

**replace(viejo, nuevo)**: Reemplaza todas las ocurrencias de una subcadena por otra.

**Sintaxis**: `cadena.replace(buscar, reemplazar)`

### Métodos de limpieza

**strip()**: Elimina espacios al inicio y final  
**lstrip()**: Elimina espacios al inicio  
**rstrip()**: Elimina espacios al final  

### Características importantes

**🔹 Retornan nueva cadena**: No modifican la original (inmutabilidad)  
**🔹 Encadenables**: Se pueden usar varios métodos seguidos  
**🔹 Case-sensitive**: Distinguen mayúsculas y minúsculas  

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### lower() y upper()

<Code code={`# Convertir a minúsculas

# Convertir a mayúsculas

mayusculas = texto.upper()
print(mayusculas)  # PYTHON ES GENIAL

# Comparación insensible a mayúsculas
usuario = input("Usuario: ")
if usuario.lower() == "admin":
    print("Acceso de administrador")

# Normalizar entrada
email = input("Email: ").lower()
print(f"Email registrado: {email}")`} lang="python" title="lower() y upper()" />

### capitalize() y title()

<Code code={`# Primera letra en mayúscula

print(titulo.title())  # Aprende Python Desde Cero

# Formatear nombres
nombre = "ana maría garcía"
print(nombre.title())  # Ana María García

# Títulos de libros
libro = "el arte de la programación"
print(libro.title())  # El Arte De La Programación`} lang="python" title="capitalize() y title()" />

### replace()

<Code code={`# Reemplazar texto

# Reemplazar múltiples ocurrencias

print(nueva)  # un gato y un perro y un pájaro

# Eliminar caracteres

### strip(), lstrip(), rstrip()

<Code code={`# Eliminar espacios en blanco

print(texto.lstrip())  # "Python   "

# Eliminar caracteres específicos

# Limpiar datos

print(limpio)  # 100`} lang="python" title="strip()" />

### Métodos encadenados

<Code code={`# Combinar múltiples métodos

print(email_limpio)  # user@gmail.com`} lang="python" title="Encadenados" />

## 🔍 10.4 Búsqueda de texto: in, find(), startswith()

<Tabs>
<TabItem label="👔 Parte Teórica">

### Operador in

**in**: Verifica si una subcadena está contenida en otra (retorna `True`/`False`)

**Sintaxis**: `subcadena in cadena`

### Método find()

**find(subcadena)**: Busca la primera ocurrencia y retorna su índice, o `-1` si no se encuentra.

**Sintaxis**: `cadena.find(buscar)`

### Métodos de verificación

**startswith(prefijo)**: Verifica si la cadena comienza con un prefijo  
**endswith(sufijo)**: Verifica si la cadena termina con un sufijo  

### Método count()

**count(subcadena)**: Cuenta cuántas veces aparece una subcadena.

**Sintaxis**: `cadena.count(buscar)`

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### Operador in

<Code code={`# Verificar si contiene subcadena

### find()

## ✂️ 10.5 Manipulación de texto con split() y join()

<Tabs>
<TabItem label="👔 Parte Teórica">

### Método split()

**split(separador)**: Divide una cadena en una lista de subcadenas usando un separador.

**Sintaxis**: `cadena.split(separador)`  
**Por defecto**: Si no se especifica separador, usa espacios en blanco

### Método join()

**join(iterable)**: Une elementos de una lista en una sola cadena usando un separador.

**Sintaxis**: `separador.join(lista)`

### Relación entre split() y join()

Son operaciones inversas:
- **split()**: Cadena → Lista
- **join()**: Lista → Cadena

### Casos de uso

**split()**: Procesar CSV, separar palabras, parsear datos  
**join()**: Construir URLs, crear rutas, formatear salida  

</TabItem>

<TabItem label="💡 Ejemplos Prácticos">

### split() básico

<Code code={`# Dividir por espacios (por defecto)

palabras = texto.split()
print(palabras)  # ['Python', 'es', 'genial']

# Dividir por separador específico

partes = fecha.split("/")
print(partes)  # ['15', '01', '2024']

# Dividir por comas

lista_frutas = frutas.split(",")
print(lista_frutas)  # ['manzana', 'pera', 'uva', 'naranja']

# Procesar entrada del usuario
entrada = input("Ingresa números separados por espacios: ")
numeros = entrada.split()
print(f"Ingresaste {len(numeros)} números")`} lang="python" title="split()" />

### join() básico

<Code code={`# Unir lista con espacios

print(texto)  # Python es genial

print(ruta)  # usuarios/documentos/python`} lang="python" title="join()" />

<Code code={`# Formatear nombre completo

print(f"{'='*40}")

print(f"Total de caracteres: {total_caracteres}")
print(f"Caracteres sin espacios: {total_caracteres_sin_espacios}")
print(f"Palabra más larga: {palabra_mas_larga} ({len(palabra_mas_larga)} letras)")
print(f"Palabra más corta: {palabra_mas_corta} ({len(palabra_mas_corta)} letras)")
print(f"Promedio de letras por palabra: {total_caracteres_sin_espacios/total_palabras:.1f}")
print(f"{'='*40}")`} lang="python" title="Analizador completo" />
